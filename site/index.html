Quickstart

How to use Android phone as webcam on Linux

Install and launch

sudo apt install ./avream_latest_amd64.deb
avream-ui

On first launch, click Enable AVream Service if lock screen appears.

Connect and start

  • Connect phone via USB and accept USB debugging prompt.
  • Click Scan Phones, select your device, then Use Selected Phone.
  • Click Start Camera.
  • Select AVream Camera (and optionally AVream Mic) in your app.
CLI

Automate and script your setup

avream status
avream devices
avream start --mode wifi --lens front
avream camera stop
