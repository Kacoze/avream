AVream command line reference

Scriptable control for status, devices, camera, Wi-Fi, and microphone.

Common flow

avream status
avream devices
avream start --mode wifi --lens front
avream camera stop

Camera

avream camera start --serial <ADB_SERIAL_OR_IPPORT> --lens front --preview-window
avream camera stop
avream camera reset

Wi-Fi

avream wifi setup --serial <USB_SERIAL> --port 5555
avream wifi connect 192.168.1.10:5555
avream wifi disconnect 192.168.1.10:5555

Microphone

avream mic start --backend pipewire
avream mic stop

JSON and socket path

avream --json status
avream --socket-path /tmp/avream.sock status

If command is missing

sudo apt install --reinstall ./avream_latest_amd64.deb
rehash
