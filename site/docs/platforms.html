Supported platforms

Stable targets and runtime requirements for AVream.

Stable targets

  • Ubuntu 22.04 (GNOME)
  • Ubuntu 24.04 (GNOME)
  • Debian 12

Architecture: amd64.

Required usage profile

  • User systemd services available in desktop session.
  • Android phone with Developer Options and USB debugging.
  • Apps that accept standard V4L2 camera and Pulse/PipeWire mic devices.

Runtime tools

  • scrcpy and adb for Android camera backend.
  • pulseaudio-utils or pw-loopback for audio virtual mic path.
