Troubleshooting

Fix common AVream issues fast

Use these recovery steps before deeper debugging.

Daemon unreachable

systemctl --user daemon-reload
systemctl --user enable --now avreamd.service
systemctl --user status avreamd.service

No phone detected

  • Use data-capable USB cable.
  • Enable USB debugging.
  • Accept phone authorization prompt.
avream devices
adb devices

Wi-Fi setup fails

avream wifi setup --serial <USB_SERIAL> --port 5555
avream wifi connect <IP>:5555

No AVream Camera in app

avream camera stop
avream camera reset
avream camera start --lens front

Mic missing

Start camera first, then pick AVream Mic in your app.

Preview not visible

Preview is a separate scrcpy window. Install scrcpy and restart AVream.

sudo apt install scrcpy
