Use Android phone as webcam on Linux

GUI flow designed for Zoom, Meet, OBS, and other Linux apps.

Camera flow

  1. Enable Developer Options and USB debugging on your phone.
  2. Connect USB, unlock phone, and accept debugging authorization.
  3. Open avream-ui.
  4. If daemon lock appears, click Enable AVream Service.
  5. Click Scan Phones.
  6. Select phone and click Use Selected Phone.
  7. Choose lens and click Start Camera.
  8. Select AVream Camera in your app.

Microphone flow

Microphone follows camera lifecycle. Start camera, then select AVream Mic in your app.

Wi-Fi mode

  1. Select USB-connected phone.
  2. Switch connection mode to Wi-Fi and click Use Selected Phone.
  3. After endpoint appears, you can disconnect USB.
avream wifi setup --serial <USB_SERIAL> --port 5555
avream start --mode wifi --lens front

Passwordless mode (optional)

avream-passwordless-setup status
pkexec avream-passwordless-setup enable --user "$USER"
