Install

Install AVream on Linux

Recommended path for Debian and Ubuntu-based systems.

Monolithic package (recommended)

sudo apt install ./avream_latest_amd64.deb

This package includes daemon, GUI, helper, and desktop integration.

Enable daemon service once

mkdir -p ~/.config/avream
cp -n /usr/lib/systemd/user/avreamd.env ~/.config/avream/avreamd.env
systemctl --user daemon-reload
systemctl --user enable --now avreamd.service

Split packages (advanced)

Release assets provide split Debian packages as one archive:

avream-deb-split_latest_amd64.tar.gz

tar -xzf avream-deb-split_latest_amd64.tar.gz
sudo apt install ./avream-daemon_latest_amd64.deb \
  ./avream-ui_latest_amd64.deb \
  ./avream-helper_latest_amd64.deb

Verify

systemctl --user status avreamd.service
avream --help
avream-ui --help

If daemon is unavailable, open GUI and use Enable AVream Service.

Uninstall

sudo apt remove avream
